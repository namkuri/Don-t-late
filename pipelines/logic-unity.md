## 3. 자동교정 루프 — 이 파이프라인의 심장
```
구현 → 컴파일 → 에러? → 에러 메시지 읽기 → 수정 → 재컴파일   (cap 2)
     → 콘솔 검사 → 에러/워닝? → 원인 수정 → 재검               (cap 2)
     → Play 검증 → 기대 불일치? → 봉투 재확인 → 수정 → 재검     (cap 2)
```
- 각 단계 2회 실패 → 부모 에스컬레이션(증상+시도+가설). "컴파일은 되는데 안 돌아요" 반환 금지.
- 성공 반환 = `변경 파일 + 확인 방법` 요약 1줄. 전체 diff·로그 나열 금지.
- **프리커밋 훅(레일, 준비기간 설치)**: 컴파일 실패·콘솔 에러 상태의 커밋을 구조적으로 차단.
  훅에 막히면 부탁이 아니라 강제다 — 고칠 때까지 저장이 안 된다.
