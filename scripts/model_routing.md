# model_routing.md — op 유형 → 모델 배정표 (v0 · 미실측)

> ⚠ **전 행 미실측** — 아래 배정은 사전 추정 규칙이다. 배정 실험(동일 op를 후보 모델로
> 교차 발주 → 리드타임·재시도·반려율 비교) 후 "근거" 칸을 실측으로 채워 확정한다.
> 리드타임 소스: `python scripts/leadtime_report.py` → planning/calibration.md.

## 배정 규칙

| op 유형 | 예 | 모델 | 근거 |
|---|---|---|---|
| 기계적 op — 형식 변환·대장 append·파일 이동·목록 집계 | new_order 스켈레톤 발주, manifest 행 추가, 스크린샷 수집 | **haiku 후보** | 미실측 — 판단 없는 정형 작업이라 최저 비용 가설 |
| 기계 판정 op — 채점기 실행·수치 대조·체크리스트 검수 | scene_stats 예산 대조, reviewer의 수용기준 대조 | **haiku 후보** | 미실측 — 판정 기준이 기계식(수치·존재 여부)일 때만. 기준 해석이 필요하면 상향 |
| 표준 구현 — 매니페스트에 있는 스크립트·View·SO, 명세가 5칸으로 닫힘 | P2~P3 스크립트 납품, 훅/채점기 제작 | **기본(sonnet)** | 미실측 — 현 세션들 기본 모델로 반려율 허용 범위였다는 인상뿐, 수치 없음 |
| 판정·설계 op — 아키텍처 결정, 반려 원인 분석, cast 발주문 작성, 동결 문서 개정안 | orchestrator 관제, oracle 근본원인, prompt-refiner | **상향(opus)** | 미실측 — 되돌리기 비용이 모델비를 지배한다는 가설 |
| 시각 판정 — 스크린샷 룩 판정(팔레트 밀착·무드) | 밤거리 컷 승인 | **사람** | 모델 배정 대상 아님 — 아트 일관성·손맛은 사람 게이트(reviewer 규약) |

## 운용 메모 (미실측 상태의 안전규칙)

- 애매하면 **한 단계 상향** — 반려 1회의 왕복이 모델비 차액보다 비싸다(가설).
- 같은 op가 2회 반려되면 모델 탓하기 전에 봉투(5칸) 품질을 의심 — prompt-refiner 경유.
- 배정 실험 기록은 이 표의 "근거" 칸에 calibration.md 행 참조로 남긴다.
